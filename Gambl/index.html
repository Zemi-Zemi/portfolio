Gambl Discord Bot 🎲

Gambl is a full-featured economy and gambling Discord bot with leveling, coin system, shop, inventory, achievements, and more.

🔗 GitHub Repository

github.com/imattas/gambl-discord-bot

🧠 Features

  • 🎰 Gambling games: Blackjack, Coinflip, Roulette
  • 💰 Economy system: Wallet, Bank, Shop, Inventory
  • 🏆 Leveling system with roles and leaderboard
  • 🎟️ Giveaways, Achievements, Badges
  • ⚙️ Admin commands for full control

🛠️ Built With

Python, Discord.py, SQLite, App Commands
